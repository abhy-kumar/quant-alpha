## <img src="https://unpkg.com/lucide-static@0.321.0/icons/target.svg" width="24" height="24" align="top" /> Goal & Impact

The primary goal of the Quantitative Alpha platform is to democratize institutional-grade quantitative strategies. It seamlessly synthesizes momentum signals, technical breakouts, and fundamental value scores into a high-performance, zero-latency interface. 

The impact of this infrastructure is the elimination of emotional bias from equity research. By bridging the gap between raw financial data and actionable trading insights, it equips the Alpha Research and Investment Club to make objective, data-driven, and mathematically rigorous investment decisions.

## <img src="https://unpkg.com/lucide-static@0.321.0/icons/zap.svg" width="24" height="24" align="top" /> Core Features

- **Algorithmic Signal Generation**: Automated scoring using RSI, MACD, Bollinger Bands, and Supertrend.
- **Deep Fundamental Screening**: Real-time integration of P/E ratios, ROE, Debt-to-Equity, and EBITDA metrics.
- **Natural Language Processing (NLP)**: Automated news fetching and VADER sentiment analysis for real-time market sentiment scoring.
- **Dynamic Charting**: Interactive price and volume charts natively integrated for seamless technical analysis.
- **Performance Optimized Architecture**: Strict market-hour polling constraints and local data caching ensuring minimal infrastructure costs and maximum uptime.

## <img src="https://unpkg.com/lucide-static@0.321.0/icons/layers.svg" width="24" height="24" align="top" /> Architecture

The platform operates on a decoupled, highly efficient stack designed for maximum speed and zero maintenance costs:

- **Frontend**: React, Vite, Tailwind CSS, Recharts
- **Data Engine**: Python (`pandas`, `numpy`, `yfinance`, `vaderSentiment`)
- **Data Delivery**: Automated static JSON artifacts served directly to the client
- **Serverless API**: Vercel Serverless Functions for real-time data streaming

## <img src="https://unpkg.com/lucide-static@0.321.0/icons/terminal.svg" width="24" height="24" align="top" /> Local Setup

To deploy the platform locally, ensure Node.js and Python 3.x are installed on your environment.

### 1. Execute the Data Engine
Initialize the Python environment and run the scanner to generate the initial static datasets and caches.

```bash
# Create and activate a virtual environment
python -m venv venv

# Windows
.\venv\Scripts\activate
# Unix/MacOS
# source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Run the data extraction and NLP scanner
python scanner.py
```

### 2. Launch the Application Interface
With the `market_data.json` artifact generated, initialize the frontend.

```bash
cd frontend
npm install
npm run build
npm run dev
```

Navigate to `http://localhost:5173` to access the local deployment.
